crucible-ls-example-tests(1)
============================
:doctype: manpage

NAME
----
crucible-ls-example-tests - list names of example tests

SYNOPSIS
--------
[verse]
*crucible ls-example-tests*

DESCRIPTION
-----------
List all example tests. Example tests are those whose names match the glob
pattern "example.*".

SEE ALSO
--------
*crucible-ls-tests(1)*
